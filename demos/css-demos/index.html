Animated CSS shape-outside

Notice how the text flows around this shape. The boundary of the shape is defined using a single CSS polygon() declaration with a new round parameter.

As the corner radius dynamically animates from 0px (sharp zigzag) to 60px (smooth waves), the shape-outside property interpolates seamlessly. The text automatically re-flows in real-time, hugging the contours of the organic shape.

Previously, achieving a rounded exclusion zone required complex SVG definitions with url(#clip) references, which were incredibly tedious to build and notoriously difficult to animate via CSS. Now, a simple polygon(round ...) does the trick natively!
